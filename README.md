# Alpha Strike Unit Browser

A Streamlit app for browsing Alpha Strike unit data retrieved from [masterunitlist.info](http://masterunitlist.info/).

Filter units by type, role, era, faction, and stats. Select a unit from the table to view its full Alpha Strike card including image, stats, and era availability.

This app was mostly generated by Claude Code as a way for me to familiarize myself with the tool.

---

## Prerequisites

- [Python 3.10+](https://www.python.org/downloads/)
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

---

## Setup

### 1. Clone the repository

```bash
git clone <repo-url>
cd MUL_retrieval_browser
```

### 2. Install dependencies

```bash
uv sync
```

### 3. Generate the unit data

Run the Jupyter notebook to retrieve unit data from masterunitlist.info and produce a `unit_list.csv` file. This makes thousands of HTTP requests and **will take several hours** to complete.

```bash
uv run jupyter nbconvert --to notebook --execute "Data retrieval from MUL.ipynb" \
  --output "Data retrieval from MUL.ipynb" \
  --ExecutePreprocessor.timeout=7200
```

Alternatively, open the notebook interactively:

```bash
uv run jupyter notebook
```

### 4. Run the app

```bash
uv run streamlit run app/app.py
```

The app will be available at [http://localhost:8501](http://localhost:8501).

---

## Notes

- The app automatically picks up the most recently modified `unit_list*.csv` file in the project directory.
- If the notebook run is interrupted (e.g. due to a network timeout), re-run the command above — it will start fresh.
- A pre-existing `unit_list_20250328.csv` is included in the repository as a starting point.
